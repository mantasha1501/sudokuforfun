# sudokuforfun

A lightweight, highly responsive retro-arcade themed Sudoku puzzle application built entirely using pure HTML5, CSS3, and modern vanilla JavaScript.

## Features

- **Retro Aesthetic**: Implements an authentic CRT scanline terminal look equipped with a neon Cyberpunk-inspired palette.
- **Dynamic Real-Time Verification**: Instant validation tracking loops verify placed numbers against the solution matrix, alerting inputs automatically in red.
- **Intelligent Hint System**: Step-by-step assistant that helps isolate empty paths or re-veals target solutions when requested.
- **Notes Grid Overlay**: Built-in visual scratchpad notes feature to track minor possibilities per cell block.
- **Mobile Friendly**: Grid scaling rules naturally downsize layout items seamlessly for smaller devices and screens.

## Quick Start

1. Clone or download this repository.
2. Open `index.html` in any modern web browser to play immediately! No compile configurations required.
